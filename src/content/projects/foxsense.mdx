## The Challenge

Foxsense needed a way to accurately determine application health by analysing APIs and their dependencies, integrated with their existing monitoring infrastructure.

## What I Built

- A health monitoring service that analyses each API and its dependency chain
- Integration with the existing monitoring system for holistic application visibility
- A fully integratable user system designed for cross-service reuse
- A notification system for alerting users about application health changes

## Architecture Decisions

The service was designed with modularity in mind — the user system and notification system were built as independent, reusable components that other services in the organisation could adopt.
