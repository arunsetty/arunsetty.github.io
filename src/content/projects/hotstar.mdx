## The Challenge

Hotstar needed a unified service to manage the entire lifecycle of engineering projects — from creation through deployment — while maintaining visibility into changes across the organisation.

## What I Built

- A service handling the full spectrum of project lifecycle activities using GoCD pipelines
- Integrated GitHub APIs for everything from repo generation to content checks
- Built a state-maintained system for tracking all organisational changes
- Designed the service to be OpenAPI compliant for future CLI tool transition

## Impact

The service streamlined project management across engineering teams, reducing manual overhead and providing a single source of truth for project state.
