## The Challenge

Build a no-code system capable of creating fully functional, structural applications from scratch — driven entirely by user specifications.

## My Role

As a core developer at Broken, I've been involved in the development of the main product since its early stages. The challenge was designing a system flexible enough to handle arbitrary application structures while maintaining reliability and performance.

## Key Contributions

- Developed the core no-code engine that translates user specifications into working applications
- Architected the structural application builder from the ground up
- Worked across the full stack to ensure seamless integration between the builder interface and the generated applications

## Technologies Used

TypeScript, React, Node.js, and various internal tooling built specifically for the no-code platform.
